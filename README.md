# -analisis-mediasi-sederhana

Sebagai mahasiswa selalu mendapat tuntutan untuk membaca literatur, menyelesaikan 
tugas, bahkan tidak jarang mahasiswa juga harus menyajikan materi secara konstan. Hal 
ini menjadi penyebab mahasiswa sangat ideal untuk dijadikan subjek dalam penelitian 
prokrastinaasi akademik. Penelitian ini bertujuan untuk mengetahui hubungan antara 
coping problem based terhadap prokrastinasi akademik melalui stres akademik sebagai 
mediator. Analisis yang dilakukan dalam penelitian ini menggunakan metode mediasi 
sederhana. Penelitian dilakukan di FMIPA Universitas Negeri Yogyakarta dengan jumlah 
responden 152 subjek. Berdasarkan hasil penelitian diperoleh koefisien regresi dugaan 𝑎 =
0,4101 dan 𝑎 = −0,4086. Artinya prokrastinasi akademik berpengaruh positif terhadap
coping problem dan coping problem berpengaruh negatif terhadap stres. Hal ini berarti 
semakin besar prokrastinasi akademik maka akan semakin besar pula coping problem yang 
dialami. Diketahui juga bahwa 𝑧ℎ𝑖𝑡𝑢𝑛𝑔 adalah 1,6935 dengan 𝑝 − 𝑣𝑎𝑙𝑢𝑒 sebesar 0,0903. 
Sehingga dapat disimpulkan bahwa hipotesis diterima, yaitu stres akademik mempunyai 
pengaruh marginal terhadap coping yang berfokus pada masalah dan prokrastinasi
akademik pada mahasiswa
